## 二、协议逆向实战流程

### 第一步：抓包分析
```bash
# 启动 mitmproxy
mitmproxy -p 8080

# 或用 OpenClaw 浏览器
browser action=navigate targetUrl=目标网站
```

### 第二步：识别加密
- 找加密函数（JADX 搜索关键词）
- Hook 加密函数（Frida）
- 提取密钥/IV

### 第三步：协议还原
- 分析请求格式
- 找签名算法
- 复现客户端

### 第四步：自动化
- 编写脚本
- 集成到 OpenClaw 技能
