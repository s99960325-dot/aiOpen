# 记忆系统 + 自我进化方案

## 一、AI 记忆系统架构

### 核心概念
**LLM Memory** = 隐式知识（训练参数） + 显式存储（运行时检索）
- 突破 token 限制
- 跨会话一致性
- 持续推理能力

### 四层架构
1. **存储层**：向量数据库（Chroma/Weaviate）/ 图数据库
2. **处理层**：Embedding 模型
3. **管理层**：写入/更新/遗忘/压缩策略
4. **应用层**：检索增强生成（RAG）

### 记忆类型
- **短期记忆**：会话上下文
- **长期记忆**：向量存储
- **情景记忆**：事件序列
- **语义记忆**：知识图谱
