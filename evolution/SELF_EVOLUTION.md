## 教训记录
- grep 找不到字段 → 先确认配置是否存在
- 不瞎加配置项 → 先查 schema
- 执行命令失败 → 分步执行，不用嵌套 $()
- 报错发给用户 → 自己消化错误，只给结果
- 编造链接 → 没搜到的老老实实说没搜到，不瞎编地址
- 上下文丢了 → 实时存数据库，不靠上下文记忆
- 主动恢复上下文 → 不主动恢复避免卡顿，需要时查库
- 瞎改核心文件 → 改核心文件先告诉seven老师改什么为什么，批准了才改
