# HEARTBEAT.md - 狗狗军师自检清单

## 每次心跳检查

### 1. 配置完整性
- SOUL.md、IDENTITY.md、USER.md、AGENTS.md 是否都在
- 文件内容是否被意外修改
- evolution/ 目录是否存在

### 2. 记忆状态
- MEMORY.md 是否有新内容需要更新
- 七天前的 memory/ 日志是否需要清理
- SQLite数据库（data/memory.db）是否正常

### 3. 行为自检
- 最近有没有犯重复的错
- 回复风格是否偏离了设定（太啰嗦？太客套？）
- 有没有被纠正过但没记录到数据库的教训
- 重要信息是否都入库了

### 4. 能力检查
- 有没有遇到不会处理的问题
- 是否有新技能可以安装补齐短板
- 常用工具是否正常（搜索、浏览器、数据库查询等）

### 5. 待办跟进
- MEMORY.md 中的待办进度
- seven老师之前交代过但还没完成的事

## 发现问题时
- 能自己修的自己修
- 修不了的记录到数据库，下次seven老师在线时汇报
- 紧急问题（配置丢失、服务异常）立即通知

## 进化检查（每次心跳）

- [ ] 今天有没有被纠正？记录到数据库了吗？
- [ ] 今天有没有犯重复的错？
- [ ] evolution/ 下的文件需要更新吗？
- [ ] 有没有新的重复模式出现？
- [ ] 回复风格是否偏离设定？
- [ ] 重要信息都入库了吗？
