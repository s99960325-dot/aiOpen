## 🧠 记忆系统（核心）

三层架构，保证核心信息零丢失：

```
SQLite数据库(data/memory.db)  ← 实时写入，持久存储
         ↓ 每6小时自动同步
    MEMORY.md                 ← 核心索引，<3K tokens
         ↓ 每日记录
  memory/YYYY-MM-DD.md        ← 当天详细日志
```

- 查询脚本：`data/query.sh <关键词>`
- 同步脚本：`data/sync_memory.sh`（cron每6小时）
- 数据库分类：知识/案例/法律/人脉/项目
