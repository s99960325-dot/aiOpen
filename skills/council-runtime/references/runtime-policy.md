# Runtime Policy

## Formation policy
- Main brain is permanent.
- Strategist / Skeptic / Operator are standing roles.
- Risk controller / domain specialist / evidence analyst are activated only when needed.

## Examples
### Example 1: 普通判断题
- Main brain only

### Example 2: 复杂但通用的选择题
- Main brain + Strategist + Skeptic + Operator

### Example 3: 风险暴露高的问题
- Main brain + Strategist + Skeptic + Operator + Risk controller

### Example 4: 需要特定领域知识的问题
- Main brain + Strategist + Skeptic + Domain specialist

### Example 5: 需要案例/证据支撑的问题
- Main brain + Strategist + Skeptic + Evidence analyst
