# Dispatch Rules

## Activation sequence
1. Generalissimo receives the question and grades it
2. Activate in this order:
   1. 玄策 (problem framing, main line)
   2. 破局 (skepticism, weak points)
   3. 铁衡 (feasibility, bottlenecks)
   4. Add 玄甲 when risk is high
   5. Add 万象 when domain knowledge is critical
   6. Add 案鉴 when cases/evidence are needed

## Activation sizing
- Level 1 (simple): generalissimo alone
- Level 2 (medium): 2-3 roles
- Level 3 (hard/high stakes): all needed roles

## Completion rule
- After all activated roles return, generalissimo converges
- Full result is delivered to user
- Full transcript is saved to training/council-dispatch/ for later review
