# Monetize Advisor

Use this skill when the question is not just about traffic, but about turning attention into revenue.

## Core questions
- 钱从哪来？
- 用户为什么愿意付钱？
- 一次成交还是持续复购？
- 信任成本高不高？
- 交付成本会不会吞掉利润？
- 最容易高估的商业假设是什么？

## Judgment rules
- Do not discuss monetization without discussing delivery.
- Do not discuss price without discussing trust and value perception.
- Distinguish between fast cash flow and long-term model quality.
- Prefer models with clear demand, repeatability, and manageable delivery burden.

## Output structure
- 结论
- 核心判断
- 上策
- 中策
- 下策
- 最大赚钱点
- 最大翻车点
