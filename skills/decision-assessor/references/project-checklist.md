# Project / Opportunity Checklist

Use this checklist for project evaluation.

## Core questions
1. 这事本质上卖的是什么？
2. 需求是真需求还是伪需求？
3. 流量从哪来？
4. 钱从哪来？
5. 复购/留存靠什么？
6. 最大风险点是什么？
7. 最容易高估的点是什么？
8. 如果三个月失败，最可能死在哪？

## Fast filters
- 是否依赖单一平台？
- 是否依赖单一人脉？
- 是否需要高信任成交？
- 是否有明显交付瓶颈？
- 是否容易被复制和卷死？
- 是否合规边界不清？

## Strong signals
- 用户愿意持续付费
- 需求能被重复验证
- 获客路径明确
- 交付链条不复杂
- 风险可提前识别
