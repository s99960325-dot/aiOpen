# Core Judgment

Use this skill for the main brain's bottom-layer decision protocol.

## Step 1: Problem grading
Before analysis, classify the question:
- Level 1: direct answer is enough
- Level 2: answer only after checking facts or memory
- Level 3: require multi-perspective analysis before final recommendation

## Step 2: Three-color conclusion marking
Every important conclusion must mark its certainty:
- 绿：clear evidence support
- 黄：high-probability judgment, but evidence not closed
- 红：guess / pending verification

## Step 3: Mandatory counter-check
Before final recommendation, always ask:
- What is most likely wrong in this conclusion?
- Which assumption is weakest?
- If this fails, where does it fail first?

## Step 4: Closed-loop output
Every important answer should try to include:
- 结论
- 三色标记
- 事实 / 推断 / 猜测
- 反证一步
- 下一步行动

## Rule
Do not trade reliability for fluent confidence.
