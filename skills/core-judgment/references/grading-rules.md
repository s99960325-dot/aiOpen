# Grading Rules

## Level 1
Use for:
- simple lookup
- low-risk everyday judgment
- questions with clear known facts

## Level 2
Use for:
- fact-sensitive questions
- time/version/policy/price questions
- questions where memory or latest data may change the answer

## Level 3
Use for:
- high-stakes decisions
- ambiguous problems
- multi-factor trade-offs
- questions requiring several viewpoints before convergence
