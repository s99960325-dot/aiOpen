# 2026-03-20 学习日志

## 第11轮：私域→复购→裂变

### 核心发现
1. **LTV黄金数据**：私域成熟企业LTV比行业均值高40%；瑞幸私域复购率15%→60%，贡献80%GMV；私域用户复购周期28天，LTV是普通用户3.2倍
2. **复购预测模型**：隐形眼镜商家300万用户实战——订单属性画像（非身份画像）+30/60/90天交叉分析，点击率+20%，流失率-50%，月流失率1.5%
3. **核心顾客法则**：数量+10%→业绩+30%，频次+1次→业绩+40%。正餐核心顾客仅3-8%，理想10-20%
4. **四种私域模式**：知识专家（母婴/美妆）、专属顾问（3C/家装）、购物参谋（咖啡/快消）、兴趣同好（高端时尚/摄影）
5. **储值锁客**：充值免单模式+付费会员体系，蛋糕品牌节点充值近千万，储值用户抗公域补贴冲击
6. **AI提效**：48.82%企业计划增加AI工具预算；AI外呼节点ROI 8-10；AI图片生成替代搭景拍摄
7. **全域平台**：小红书稳居第二，视频号未来投入并列第二；企微仍是断崖领先

### 变现贯穿分析
- 私域营收占比5-30%为黄金区间，超50%风险大（监管趋严）
- 公域获客成本320元/人，留存率<25%，私域是"自建数字地产"
- 核心指标：用户活跃率>复购率>LTV>客单价，不能只盯营收
