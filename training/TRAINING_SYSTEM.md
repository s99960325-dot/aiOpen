# TRAINING_SYSTEM.md - 狗狗军师训练系统

## 目标
把狗狗军师与智囊团做成可训练、可评分、可复盘、可进化的主脑系统。

## 训练闭环
1. 选题：真实案例 / 模拟题 / 最新问题
2. 初判：主脑或智囊团给出判断
3. 标记：区分事实 / 推断 / 猜测
4. 对照：对比真实结果、标准答案、最新资料
5. 评分：看偏差、稳定性、可靠性
6. 复盘：找出错因
7. 反哺：把修正写入记忆、规则、训练文件

## 四类训练
### 1. 案例训练
- 用真实世界案例做判断
- 重点看：问题定义、主线判断、风险识别、收敛质量

### 2. 考试训练
- 用有参考答案或结果的题做测评
- 重点看：正确率、偏差类型、速度与稳定性

### 3. 搜索校验训练
- 针对最新问题，先判断，再搜索，再校验
- 重点看：旧知识偏差与新知识修正能力

### 4. 复盘训练
- 针对已做判断复盘
- 重点看：到底错在问题定义、证据、风险、收敛还是落地

## 评分维度
每次训练至少打这6项：
- 问题定性
- 证据可靠性
- 反证质量
- 风险识别
- 收敛质量
- 行动路线可执行性

## 评分等级
- 9-10：很稳，可复用
- 7-8：基本可靠，但有偏差
- 5-6：勉强可用，需要复盘
- 0-4：判断明显失真，必须纠偏

## 反哺规则
- 稳定有效规则 -> MEMORY.md
- 当前训练重点 -> NOW.md
- 关键修正与拍板 -> DECISIONS.md
- 具体训练记录 -> training/ 或 daily log
