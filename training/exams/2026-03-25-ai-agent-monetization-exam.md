# 2026-03-25 AI Agent 变现模式考试

## 题目
从最新公开信息判断，2026 年 AI Agent 变现最稳的三个方向是什么，为什么？

## 作答
- 结论：1) RaaS（结果即服务）；2) 混合定价（基础订阅 + 超额计费）；3) 席位订阅 / 能力授权。
- 三色标记：绿
- 事实：多个独立最新来源都指向 RaaS 增长最快，混合定价已经成为行业普遍落地方式，席位订阅是 B2B 侧稳定变现方式。
- 推断：纯工具订阅利润率会持续走低，纯 token 计费用户体验不稳定，纯 API 开放更适合头部玩家，不适合中小/垂直玩家。
- 猜测：未来 1~2 年 RaaS 会成为垂直 AI Agent 玩家最赚钱的路线。
- 反证一步：
  - 结论最可能错在：头部玩家能走的路线，中小玩家不一定能走。
  - 最脆前提：公开数据偏向报道头部，对中小玩家实际生存状态覆盖不足。
  - 如果失败，会死在：给中小玩家推了他们扛不动的模式。
- 下一步行动：如果做垂直 AI Agent 项目，优先从 RaaS 切入，不优先抢 token 分发赛道。
